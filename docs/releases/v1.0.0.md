# v1.0.0 - Initial public release

This is the first public release of `TencentCloudDdnsCSharp`, a Windows DDNS client for Tencent Cloud DNSPod.

## Highlights

- Supports Tencent Cloud DNSPod dynamic DNS updates for AAAA records
- Includes IPv6 address detection from local adapters and external URL providers
- Supports Windows service deployment with bundled install/start/stop scripts
- Includes unit tests and GitHub Actions workflows for build, test, and release packaging
- Ships with a safe sample configuration template and does not include local runtime secrets

## Package contents

The release ZIP contains:

- `TencentCloudDdnsCSharp.exe`
- required runtime files
- Windows service helper scripts
- safe example configuration templates from `conf/`

## Quick start

1. Download the `win-x64` ZIP asset from this release.
2. Extract it to a local directory.
3. Copy `conf/example.example.com.jsonc` to your own local `conf/<your-domain>.json`.
4. Fill in `SecretId`, `SecretKey`, domain settings, and IP provider settings.
5. Run `TencentCloudDdnsCSharp.exe -c` once in console mode to verify the update.
6. Install it as a Windows service if you want it to run in the background.

## Notes

- Do not commit real `conf/*.json` runtime files to source control.
- The sample file is a template only; active runtime config uses `conf/*.json`.
- The release package is built automatically by GitHub Actions from the tagged source.
